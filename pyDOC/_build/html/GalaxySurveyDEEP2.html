DEEP2 Survey¶

This class is dedicated to handling observations from DEEP2 galaxy redshift survey.

class GalaxySurveyDEEP2.GalaxySurveyDEEP2[source]¶

The class GalaxySurveyDEEP2 is dedicated to handling DEEP2 survey and the class GalaxySpectrumDEEP2 to handling its spectra.

class GalaxySurveyDEEP2.GalaxySurveyDEEP2(redshift_catalog='zcat.deep2.dr4.v2.fits', calibration=True)[source]
Loads the environement proper to the DEEP2 survey :
  • Defines all the proper paths in the database,
  • Opens the catalog and different calibration files,
  • Loads a list of the DEEP2 spectra.
Parameters:
  • redshift_catalog – name of the DEEP2 redshift catalog (path to the fits file)
  • calibration – if the class is loaded with intention of flux calibrating the DEEP2 data (boolean)
